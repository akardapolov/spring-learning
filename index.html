🚀 Как использовать

📖 Порядок работы с модулем

1
Выберите нужный модуль в левом меню или кликните карточку выше.
2
Изучите теорию — содержимое index.html модуля отображается прямо на этой странице.
3
При наличии исполняемого приложения запустите модуль через Maven и протестируйте доступные endpoint'ы.

🧩 ООП и паттерны в Spring

📋 Сравнительная таблица подходов

Подход / ПаттернСутьПлюсыМинусыВ Spring
НаследованиеПодкласс переопределяет методы через extendsПросто, встроено в языкЖёсткая связность, нельзя с finalОграниченно
КомпозицияКласс содержит экземпляр другого и использует егоГибче наследования, слабее связностьБольше шаблонного кода✓ Широко
ДелегированиеОбъект передаёт выполнение другому объектуГибко, хорошо с интерфейсамиМного однотипного кода✓ Широко
Wrapper / ОбёрткаОборачивает объект, повторяя его интерфейсНе меняет исходный классМного boilerplate✓ Да
ДекораторДобавляет поведение до/после вызова, тот же интерфейсХорошо расширяется цепочкойУсложняет цепочку вызовов✓ Да
Стратегия (Strategy)Поведение выносится в объект и подставляется через DIОчень гибко, легко тестироватьНужно заранее проектировать✓ Очень широко
Шаблонный методБазовый класс задаёт алгоритм, шаги переопределяютсяУдобно для переиспользованияОснован на наследовании✓ *Template
Dynamic Proxy (JDK)Прокси для интерфейса перехватывает вызовы методовБез изменения кодаТолько для интерфейсов✓ Spring AOP
CGLIB / ByteBuddyПодкласс целевого класса переопределяет методыПроксирует классы без интерфейсаНе работает с final✓ Широко
AOPВынесение сквозной логики в аспектыУбирает дублированиеПоведение менее очевидно✓ Ключевая фича
Interceptor / FilterВстраивание логики до/после обработкиMiddleware-подходНужно знать порядок цепочки✓ HandlerInterceptor
Listener / ObserverПодписчики реагируют на событияСлабая связностьПоведение распределено✓ ApplicationListener
Лямбды / CallbacksПоведение передаётся как функцияКратко, меньше классовНе для сложного состоянияОбщий стиль Java
Bytecode instrumentationИзменение байткода на летуОчень мощноСложно, низкоуровневоСторонние инструменты
